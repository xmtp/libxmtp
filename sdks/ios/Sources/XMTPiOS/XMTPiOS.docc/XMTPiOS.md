# ``XMTPiOS``

XMTP is the most secure way to add messaging to your app.

## Overview

Use the XMTP iOS SDK for a Swift interface to XMTP to build quantum-resistant group chats and direct messages that support flexible identities, built-in consent, and rich content types. XMTP implements MLS (Messaging Layer Security) for end-to-end encrypted messaging.
